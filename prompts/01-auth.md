# Screen 1: Authentication — Login / Sign Up

A polished, trustworthy authentication gateway for a modern finance-adjacent web app — dark, focused, and distraction-free.

**DESIGN SYSTEM (REQUIRED):**
- Platform: Web, Desktop-first (1280px), responsive to mobile
- Theme: Dark, glassmorphism, high-contrast with vibrant violet accent
- Background: Midnight Navy Gradient (#0f172a → #1e293b), soft diffused violet glow behind card
- Surface: Frosted Glass Card — Slate-800 at 60% opacity (#1e293b), 1px white/10% border, 16px blur
- Primary Accent: Electric Violet (#7c3aed) for CTA buttons, active tabs, focus rings
- Text Primary: Pure White (#f8fafc) for headings
- Text Secondary: Cool Slate (#94a3b8) for labels, placeholders
- Input Fields: Dark Slate (#0f172a) background, white text, violet focus ring (2px glow)
- Buttons: Fully pill-shaped (border-radius: 9999px), violet gradient primary, ghost outlined secondary
- Font: Inter — 700 heading, 400 body

**Page Structure:**
1. **Brand Header:** Centered logo (plane + receipt icon in violet), "TripLedger" in 28px bold white, tagline "Smart group expense tracking" in slate-400
2. **Auth Toggle Tabs:** "Login" | "Sign Up" side-by-side tabs, animated violet underline indicator on active tab, smooth transition
3. **Login Form:** Email input field, Password input field (with eye-icon toggle for show/hide), "Forgot Password?" right-aligned link in violet
4. **Primary CTA Button:** Full-width pill-shaped "Continue with Email" in violet (#7c3aed) with gradient shimmer
5. **Divider:** Horizontal line with "or continue with" text centered in slate-500
6. **OAuth Button:** Google sign-in ghost button — white border, Google color logo, "Continue with Google" label
7. **Footer:** "By continuing you agree to our Terms of Service and Privacy Policy" in slate-500, 12px
